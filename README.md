
# Mail Relaying over DNS by Bugfish

This is a PHP/MySQL Software which can be run in a webinterface
and with a cronjob to sync to domains registred on a secondary DNS to the Postfix Relaydomains and Transportmaps File. You can Enter Manual Domains and there is a simple User Management.

#### Installation

---------------------------------------------------------------  
* Check config.php and check Config Variables for Validity
* Check cronjob.php Path Variable on Top to fit to your systems Path
* Check if your Postfix is Setup like in Example
* See if Cronjob is  setup like in example
